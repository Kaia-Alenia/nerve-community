# Nivel 14: Archivos 💾

Todo lo que has programado hasta ahora vive en la memoria RAM. Cuando el programa termina, los datos desaparecen.

Leer y escribir **archivos** cambia eso: permite que tu programa guarde información que persiste entre ejecuciones, lea configuraciones, procese datos externos y registre eventos.

Es el primer paso hacia la persistencia de datos real.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende a abrir, leer y escribir archivos de texto.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` para ver el ciclo completo: escribir → leer → añadir.
3. ⚔️ **Supera el reto**: Abre `reto.md` y crea tu propio registro de datos en archivo.
