# Nivel 13: Manejo de Errores 🛡️

En el nivel anterior, si el usuario escribe `"hola"` cuando pedías un número, el programa explota con un `ValueError` y muere. En una aplicación real, eso es inaceptable.

El **manejo de errores** te permite anticiparte a los fallos y decirle al programa qué hacer cuando algo sale mal, en lugar de simplemente caerse.

Es la diferencia entre una app profesional y un programa frágil.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende la estructura `try / except`.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` y observa cómo el programa sobrevive a entradas incorrectas.
3. ⚔️ **Supera el reto**: Abre `reto.md` y blinda tu propio programa.
