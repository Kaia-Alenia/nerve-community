# Nivel 12: Input del Usuario ⌨️

Hasta ahora, todos los datos de tus programas los escribiste tú directamente en el código. Pero las aplicaciones reales son interactivas: le **preguntan** al usuario qué quiere.

La función `input()` pausa el programa, muestra un mensaje y espera a que el usuario escriba algo y presione Enter. Con esto, tus programas cobran vida propia.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende a capturar datos del usuario.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` y observa la trampa del tipo de dato.
3. ⚔️ **Supera el reto**: Abre `reto.md` y crea tu primer programa interactivo.
