# Nivel 11: Funciones 🧰

Hasta ahora, cada vez que necesitabas repetir una tarea, copiabas el código. Si lo necesitabas en 10 sitios distintos, lo copiabas 10 veces. Eso es un caos: si hay un error, lo tienes que corregir 10 veces.

Las **funciones** resuelven eso: defines el código una sola vez, le pones un nombre, y lo llamas cuantas veces quieras desde cualquier parte del programa.

En el mundo real, cada botón de una aplicación llama a una función. Sin funciones, no existe ningún software moderno.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende a definir y llamar funciones.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` para ver cómo las funciones organizan el código.
3. ⚔️ **Supera el reto**: Abre `reto.md` y escribe tus propias funciones.
