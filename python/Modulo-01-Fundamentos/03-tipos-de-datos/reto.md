# Reto 03: Ficha de Personaje 🎮

Es hora de crear tu propia ficha de personaje de videojuego. Deberás guardar la información correcta utilizando el tipo de dato adecuado.

## Instrucciones

1. Dentro de la carpeta `solucion/` que está en `03-tipos-de-datos/`, crea tu archivo con el formato `tu-usuario-03.py` (por ejemplo, `kaia-03.py`).
2. Crea una variable llamada `alias` y asígnale el nombre de tu personaje como **Texto**.
3. Crea una variable llamada `nivel_actual` y asígnale el número `10` como un **Número Entero** (sin comillas).
4. Crea una variable llamada `nivel_proximo`. En esta caja vas a guardar el resultado de sumarle `1` a tu `nivel_actual` (usa el símbolo `+`).
5. Crea una variable llamada `modo_invencible` y asígnale el estado falso como un **Booleano**.
6. Usa varios `print()` intercalados con separadores `---` (tal como vimos en el ejemplo) para mostrar toda esta información en tu terminal. Imprime el `alias`, luego el `nivel_actual`, el `nivel_proximo` y finalmente el `modo_invencible`.

**Resultado esperado en tu terminal:**
```text
--- Mi Personaje ---
Kaia
--- Nivel Actual ---
10
--- Nivel Siguiente ---
11
--- ¿Es Invencible? ---
False
```

*(Tu resultado tendrá el nombre que tú hayas elegido, ¡pero asegúrate de que los números y el estado `False` se vean tal cual!)*

Ejecuta tu código desde la raíz del proyecto:
- **macOS / Linux:** `python3 python/Modulo-01-Fundamentos/03-tipos-de-datos/solucion/tu-usuario-03.py`
- **Windows:** `python python/Modulo-01-Fundamentos/03-tipos-de-datos/solucion/tu-usuario-03.py`

Si lograste imprimir toda esta información de forma estructurada y Python no se quejó de ningún error... ¡Estás superando las expectativas y listo para el próximo reto!
