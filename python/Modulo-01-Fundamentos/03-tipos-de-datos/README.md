# Nivel 03: Tipos de Datos 🧬

Hasta ahora, hemos guardado texto en nuestras cajas (variables). Pero las computadoras son calculadoras gigantes; necesitan saber si lo que guardamos es texto, un número para hacer matemáticas, o un simple interruptor de encendido/apagado.

En este nivel descubriremos cómo Python clasifica la información. Entender esto evitará que intentes sumar letras o multiplicar palabras por error.

## Ruta de aprendizaje

Para superar este nivel, sigue estos pasos en orden:

1. 📖 **Lee la teoría**: Abre el archivo `teoria.md` para entender las tres naturalezas básicas de la información.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` y ejecútalo para ver los tipos de datos en acción en tu terminal.
3. ⚔️ **Supera el reto**: Abre `reto.md` y demuestra que sabes clasificar tus datos.
