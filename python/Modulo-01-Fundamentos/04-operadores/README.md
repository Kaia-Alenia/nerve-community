# Nivel 04: Operadores ➕➖✖️➗

Hasta ahora, nuestras "cajas" solo guardan datos quietos. Hoy les damos vida: vamos a **operar** con ellos.

Un **operador** es un símbolo especial que le dice a Python "haz esto con estos datos". Existen dos familias que conoceremos hoy:

- **Aritméticos**: hacen matemáticas con números (`+`, `-`, `*`, `/`, `//`, `%`).
- **De comparación**: responden preguntas de Sí/No entre dos valores (`==`, `!=`, `>`, `<`, `>=`, `<=`).

Esta lección es el motor de toda lógica futura. Sin operadores, un programa no puede tomar decisiones.
