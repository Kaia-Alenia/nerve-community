# Nivel 05: Condicionales 🔀

Hasta ahora, nuestros programas son como trenes en un riel: siempre van en la misma dirección, línea por línea, sin importar nada.

Hoy les damos **cerebro**: aprenderán a **tomar decisiones**.

Con los condicionales, el programa puede preguntar "¿esto es verdad?" y elegir un camino diferente según la respuesta. Esto es exactamente lo que hace tu aplicación de mensajería cuando pregunta "¿Ingresaste la contraseña correcta?" y decide si te deja entrar o no.

## Ruta de aprendizaje

Para superar este nivel, sigue estos pasos en orden:

1. 📖 **Lee la teoría**: Abre `teoria.md` para entender la estructura `if` / `else`.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` y presta atención especial a los **espacios** (la indentación).
3. ⚔️ **Supera el reto**: Abre `reto.md` y escribe tu propia lógica de decisión.

> ¡Advertencia! Este nivel introduce el concepto más importante de programación: la **indentación**. Un solo espacio de más o de menos rompe todo. Lee con calma.
