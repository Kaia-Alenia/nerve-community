# Nivel 07: Tuplas 🔒

Ya conoces las listas: colecciones que puedes modificar, añadir y eliminar elementos.

Pero a veces necesitas una colección que **nadie pueda cambiar accidentalmente**: las coordenadas de un mapa, los días de la semana, las constantes de un sistema. Para eso existen las **Tuplas**.

Una tupla es como una lista **sellada con candado**. Puedes leer su contenido, pero no puedes modificarlo una vez creada.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende la diferencia clave entre tuplas y listas.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` para ver cuándo usar cada una.
3. ⚔️ **Supera el reto**: Abre `reto.md` y demuestra que sabes elegir la estructura correcta.
