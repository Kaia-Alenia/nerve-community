# Nivel 10: Bucle while ⏳

El `for` repite código **un número conocido de veces** (una vez por cada elemento de una lista).

Pero ¿qué pasa cuando **no sabes cuántas veces** se repetirá? Por ejemplo: "sigue pidiendo la contraseña mientras el usuario se equivoque" o "sigue disparando mientras el personaje tenga munición".

El **bucle `while`** repite un bloque de código **mientras** una condición sea `True`. Se detiene cuando la condición se vuelve `False`.
