# Nivel 09: Bucle for 🔁

Tienes una lista de 100 productos y necesitas imprimir cada uno. ¿Escribes 100 líneas `print()`? Nunca.

El **bucle `for`** le dice a Python: "repite estas instrucciones una vez **por cada elemento** de esta colección". Es la herramienta más poderosa que has aprendido hasta ahora.

Con el `for`, lo que tomaba 100 líneas de código se reduce a 2.
