# Nivel 06: Listas 📋

Hasta ahora, cada variable guardaba **un solo dato** a la vez. Pero ¿qué pasa si necesitas guardar los nombres de 100 jugadores? ¿Creas 100 variables?

Las **Listas** resuelven ese problema: son una única "caja" que puede contener **múltiples datos en orden**, uno detrás del otro, como los vagones de un tren.

Las listas son la estructura de datos más usada en Python. Las encontrarás en todo: resultados de búsquedas, filas de base de datos, feeds de redes sociales.

## Ruta de aprendizaje

1. 📖 **Lee la teoría**: Abre `teoria.md` y aprende a crear listas y acceder a sus elementos.
2. 🔬 **Estudia el ejemplo**: Revisa `ejemplo.py` para ver cómo manipular una lista paso a paso.
3. ⚔️ **Supera el reto**: Abre `reto.md` y construye tu propia lista desde cero.
